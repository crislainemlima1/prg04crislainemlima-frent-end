# 📚 FocusFlow — Plataforma Inteligente de Estudos com IA

> Transforme sua rotina de estudos com tecnologia, produtividade e Inteligência Artificial.

---

## 🧠 Sobre o Projeto

O **FocusFlow** é uma plataforma web desenvolvida para transformar sessões de estudo dispersas em uma rotina produtiva e mensurável. Reunindo em um só lugar tudo que um estudante precisa, a plataforma combina o método Pomodoro, organização por matérias, acompanhamento de evolução e o poder da IA para gerar resumos automáticos de conteúdos.

O projeto nasceu como trabalho da disciplina de **Programação Web** e está em constante evolução, com planos de expansão para integração completa com backend e deploy em produção.

## ✨ Funcionalidades

- ⏱️ **Timer Pomodoro** — ciclos de foco e descanso totalmente configuráveis
- 📊 **Dashboard de Progresso** — gráficos de horas estudadas por matéria e streak diário
- 📚 **Gerenciamento de Matérias** — cadastre disciplinas e defina metas de estudo
- 🤖 **Resumos com IA** — cole qualquer conteúdo e gere um resumo automático
- 🏆 **Gamificação** — conquiste badges e acompanhe sua evolução
- 🔐 **Autenticação** — fluxo completo de login com painel administrativo
- 📱 **Design Responsivo** — adaptado para smartphones, tablets e desktops

## 🗂️ Estrutura do Projeto

O repositório conta com duas versões do projeto:

### Versão 1 — Protótipo (HTML/CSS/JS)
Desenvolvida ao longo das aulas, serviu como base para o aprendizado de estruturação semântica, estilização e responsividade.

### Versão 2 — Refatoração (React)
Migração completa para React com componentização, roteamento via React Router DOM e arquitetura escalável.
